## HAUT_autoCheck

**河南工业大学完美校园健康打卡**
* 随机温度(36.2℃-36.8℃)🌡
* 每天定时打卡（默认早上7点）⏱
* 微信通知打卡结果(可选)💬
* 基于Github Actions，完全解放你的设备的服务器。✔
* 有任何问题可以提交[issues](https://github.com/YooKing/HAUT_autoCheck/issues/new)
